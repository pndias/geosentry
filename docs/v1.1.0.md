# GeoSentry - UX Update and Data Expansion (v1.1.0)

## 📅 Date: March 16, 2026
## 🚀 What's New:

### 1. UX: Source Transparency
- **Source Links in Modals**: Each event marker on the map now has a direct "View Original Source" link in the popup. This allows users to easily validate information, increasing dashboard transparency and credibility.

### 2. Intelligence Expansion (Mockup Data Bank)
- **Data Scale**: The local data generator (`seed_db.py`) was massively expanded. It now generates **500 recent events** (previously 100), filling the map and stress-testing rendering capacity more rigorously.
- **North Global Inclusion (USA and Europe)**: Added new base events focused on political and economic decisions from the United States and Europe (e.g., US Treasury Sanctions, Fed Interest Rates, NATO Movements, European Parliament Resolutions). This directly reflects GeoSentry's philosophy: monitoring how North Global decisions affect and reverberate in the Global South.

### 3. Database and Frontend Improvements
- The TypeScript `Event` interface now reflects correct typing to handle `source_link`.
- Modals and popups visually optimized to contain links cleanly and accessibly.
