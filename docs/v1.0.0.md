# GeoSentry - Versão 1.0.0 (Gênese)

## 📅 Data: 08 de Março de 2026
## 🏛️ Propósito
GeoSentry é uma plataforma de monitoramento geopolítico autônomo desenhada para fornecer clareza, preparação e uma visão analítica sobre conflitos e tratados globais. O objetivo final é servir como uma ferramenta para a preservação da vida e busca pela paz através da inteligência de dados.

## 🤖 Prompt de Origem (The Spark)
> "Atue como Engenheiro de IA. Crie um script Python usando o framework CrewAI. Defina dois agentes: 1) 'Pesquisador Geopolítico', focado em buscar notícias recentes sobre tratados e conflitos militares. 2) 'Analista de Dados', que formata os achados do pesquisador em JSON. Crie as tarefas para cada um e o código de execução do 'Crew'. Retorne apenas o código Python funcional."

## 🏗️ Arquitetura (V1.0.0)
Nesta versão inicial, o projeto foi estruturado seguindo as melhores práticas de modularidade e extensibilidade:
- **`src/core`**: Modelos de dados centralizados (Pydantic).
- **`src/api`**: Servidor FastAPI para distribuição de eventos.
- **`src/agents`**: Orquestração de agentes autônomos via CrewAI.
- **`src/ingestion`**: Lógica de processamento e estruturação de dados (LLM Ready).
- **`frontend`**: Interface geográfica interativa via Leaflet.js.
- **`docker`**: Conteinerização para PostgreSQL/PostGIS e serviços.

## 🛠️ Tecnologias
- Python 3.11+
- FastAPI
- CrewAI
- Pydantic V2
- Leaflet.js
- Docker & Docker Compose
