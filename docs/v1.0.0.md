# GeoSentry - Documentação de Gênese (v1.0.0)

## 📅 Data: 09 de Março de 2026
## 🏛️ Propósito e Filosofia
GeoSentry é um "Escudo de Dados" projetado para o Sul Global. Sua missão é utilizar Inteligência Artificial para salvar vidas através do monitoramento geopolítico autônomo, priorizando a soberania popular, a segurança civil e a perspectiva da classe trabalhadora em cenários de conflito.

## 🛠️ O que foi construído hoje:

### 1. Arquitetura Modular (Microserviços)
O projeto foi totalmente reestruturado para suportar escalabilidade e pluralidade:
- **`src/core`**: Modelos de dados rigorosos (Pydantic) que garantem a integridade da informação entre todos os serviços.
- **`src/api`**: Backend em FastAPI com endpoints assíncronos e suporte a CORS.
- **`src/agents`**: Orquestração de Agentes IA (CrewAI) com foco em OSINT (Open Source Intelligence).
- **`src/ingestion`**: Módulos de processamento e extração de dados estruturados.

### 2. Inteligência Artificial com Propósito
- **Agentes Sentinelas**: Implementação de agentes de IA treinados para buscar notícias reais via `SerperDevTool`.
- **Diretriz Ética**: Prompts refinados para priorizar o impacto humano e social, ignorando vieses puramente financeiros ou elitistas.

### 3. Frontend de Próxima Geração (React + TypeScript)
- **Transição Tecnológica**: Evolução de um protótipo estático para uma SPA (Single Page Application) moderna em React.
- **Dashboard Responsivo**: Design "Command Center" com sidebar de eventos, estatísticas em tempo real e filtros por categoria.
- **Visualização Geoespacial**: Integração avançada com `Leaflet` e `react-leaflet` para plotagem dinâmica de eventos.

### 4. Governança e Transparência
- **MANIFESTO.md**: Documento de princípios que dita a conduta ética e o compromisso do projeto com a vida e a pluralidade honesta.
- **ROADMAP.md**: Plano estratégico dividido em 4 fases, desde a fundação até o módulo de semiótica comparada.

### 5. Infraestrutura e Segurança
- **Conteinerização**: Dockerfiles e Docker Compose preparados para orquestrar API, Banco de Dados PostGIS e Frontend.
- **Segurança de Credenciais**: Implementação de `.env.example` para garantir que segredos de API nunca sejam expostos.
