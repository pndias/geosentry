# GeoSentry - Documentação de Gênese (v1.0.0)

## 📅 Data: 09 de Março de 2026
## 🏛️ Propósito e Filosofia
GeoSentry é um "Escudo de Dados" projetado para o Sul Global. Sua missão é utilizar Inteligência Artificial para salvar vidas através do monitoramento geopolítico autônomo, priorizando a soberania popular, a segurança civil e a perspectiva da classe trabalhadora em cenários de conflito.

## 🛠️ O que foi construído hoje:

### 1. Arquitetura Modular (Microserviços & Clean Architecture)
- **`src/core`**: Modelos de dados rigorosos (Pydantic).
- **`src/api`**: Backend refatorado com rotas (`routers`) e lógica de negócios (`services`) isoladas.
- **`src/agents`**: Orquestração de Agentes IA (CrewAI) com foco em OSINT via Serper.
- **`frontend`**: SPA moderna em React + TypeScript, componentes isolados e Custom Hooks.

### 2. Dashboard GeoSentry v1.1
- **Identidade**: Atualizado para "Sentinela do Sul Global" (Avatar: GS).
- **Visualização**: Mapa interativo com marcadores dinâmicos por impacto (1-5) e categorias coloridas.
- **Responsividade**: Layout adaptável com sidebar de eventos e estatísticas.

### 3. Persistência de Dados (SQLite Seed)
- Implementação inicial de banco de dados via **SQLAlchemy** (`sqlite:///./geosentry.db`).
- Script de seed (`seed_db.py`) gerando **100 ocorrências realistas e randômicas** focadas no Sul Global para testes de carga e renderização do mapa.
- API totalmente conectada e lendo dados do banco real.

### 4. Governança e Transparência
- **MANIFESTO.md**: Princípios de soberania tecnológica, proteção à vida e defesa da classe trabalhadora.
- **ROADMAP.md**: Plano de 4 fases para o futuro do sistema, incluindo análise PLN.

### 5. Estabilidade e Correções (Frontend)
- **TypeScript e Build**: Correção das configurações do compilador (`tsconfig.json`) ativando `esModuleInterop` e `allowSyntheticDefaultImports` para estabilizar os imports do React.
- **Formatação de Código**: Substituição de comentários padrão Python (`#`) por comentários padrão JavaScript (`//`) nos arquivos `.ts` e `.tsx` para evitar quebras no parser do Babel/Vite.
- **Copyright**: Proteção legal ativada com Licença MIT e atribuição de autoria a Pablo Dias, aplicada de forma segura (sem quebrar a compilação).
