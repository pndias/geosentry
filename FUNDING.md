## 📋 Grant Application Core Narrative

### Problem Statement
Communities in the Global South lack access to real-time, structured geopolitical intelligence. Existing tools (Janes, Crisis24, ACLED) are expensive, Western-centric, and inaccessible to local NGOs, journalists, and civil society organizations operating in conflict-affected regions.

### Solution
GeoSentry provides autonomous, AI-driven geopolitical monitoring with:
- **OSINT agents** that scan and classify global events in real time
- **Structured data** (validated JSON schemas) enabling programmatic access
- **Geospatial visualization** for situational awareness
- **Open-source architecture** allowing local adaptation and self-hosting

### Impact Metrics
- Number of events monitored and classified per day
- Geographic coverage (countries/regions actively tracked)
- Downstream users (NGOs, journalists, researchers) accessing the platform
- Community contributions and local deployments

### Differentiators
1. **Global South perspective by design** — not an afterthought
2. **Fully open-source** — communities can self-host and adapt
3. **Local LLM support (Ollama)** — works without cloud API dependencies
4. **Offline-capable roadmap** — designed for low-bandwidth conflict zones
